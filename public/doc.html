Authentication

POST /api/auth/register
Register a new user

Request Body

Parameter Type Required Description
email string Yes User's email address
password string Yes User's password (min 8 characters)
name string No User's full name

Example Request

POST /api/auth/register Content-Type: application/json { "email": "user@example.com", "password": "securepassword123", "name": "John Doe" }

Success Response (201)

{ "success": true, "data": { "user": { "id": "60d21b4667d0d8992e610c85", "email": "user@example.com", "name": "John Doe" }, "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..." } }
